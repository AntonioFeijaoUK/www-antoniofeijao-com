Linux find command

!!! warning
    Use the flag `-exec` with care.
    Try the command find without the `-exec rm {} \;` to see which files are found,

```bash

find . -type f -name 'desktop.ini' -exec rm {} \;

find . -type f -name '.DS_Store' -exec rm {} \;

```
